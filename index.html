Just Do It

Welcome to this website. This website is created solely for BWEP assignment.

For this assignment, our group was instructed to create a website which allows people(users) to customize and make their own invitations based on three categories: Informal, Formal and Corporate.

The information below will tell the users what they would be able to customize:

Name

The users will be able to insert their names in the invitation generator.

Color

The users are able to change the colors of the invitations created in the invitation genrator.

Design

The designs are up to the users to pick from the design we've prepared for them.
